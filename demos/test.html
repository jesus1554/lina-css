About

Lina CSS is born from a need that all web developers have: to make web pages quickly. That is why I started working on Lina CSS, in fact, I relied on a library that I really liked (Flat Remix CSS). This library is simple, with nice components and easy to implement. Unlike other libraries, Lina CSS focuses on being as practical and useful as possible, with easy-to-remember components and intuitive options.

Lina includes commonly used web components to save the most amount of time when developing, and it also includes Normalize so that your websites can be seen well in any browser.

This project was born in the development of another project, so Lina CSS will be updated quite often because I still don't feel like it is completely finished.

 </div>

Center Elements

This class can center elements horizontally and vertically using Flexbox.

Horizontal 
<div class="center-center-h"> 
 <!-- Put your elements here -->
 </div>
Vertical 
<div class="center-center-v"> 
 <!-- Put your elements here -->
 </div>

Unities

Lina CSS uses REM as default unity, but don't worry, it's super easy! 1rem = 10px.

Of course you can disable it putting in your body styles the font size that you want. But why Lina CSS uses REM? For many reasons, most of them are in this video. 

.square-example-1 {
  height: 5rem;
  width: 5rem;
  background-color: #f372c0;
 }
